# socialapp

Features

    Responsive Instagram UI
    Email & Password Authentication using firebase
    Share Posts with Caption
    Display Posts with Caption
    Like & Comment on Posts
    Search Users
    Follow Users
    Display User Posts, Followers & Following
    Sign Out
